# Contributing

1. Uninstall any old versions of Python. Install Python **3.8** from [here](https://www.python.org/ftp/python/3.8.3/python-3.8.3.exe). Make sure to check "Add Python to PATH" when installing
2. Install Pyinstaller package installed via pip or conda
    - ```pip install -U pyinstaller```
    - Documentation [here](https://pyinstaller.readthedocs.io/en/v3.6/)
3. To start contributing, first clone/download the repo to your machine.
4. Make your changes
5. Open a command prompt in the same folder as your code (```Shift-RightClick, Open powershell window here```)
6. Run ```pyinstaller main.spec```
7. Test
8. Make a commit/PR
